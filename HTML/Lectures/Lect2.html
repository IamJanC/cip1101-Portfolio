Getting Started

Before You Start Creating Websites

You must first ask these questions:

  • What are the tools the professionals use?
  • What will my website look like?
  • What are the important files that I should have?
What Professionals Use
  • A Computer
  • Text Editor, to write the code in
  • Web Browsers, to test the code in
  • Graphic Editing Software to make and edit graphics on your web pages.
    Examples: Gimp, Figma, Photoshop
  • Version Control System, to manage files on servers.
    Examples: GitHub, GitLab
  • FTP program for older web hosting accounts (optional, Git is replacing FTP).
    Examples: Cyberduck, FileZilla
  • Automation System to run repetitive tasks like tests
  • Libraries, frameworks, etc.
What You Need to Prepare
  • Computer: Any specs will do, but invest in a good machine for serious work.
  • Text Editor: Default Notepad is okay, but you can start with Notepad++
  • Web Browser: Chrome, Firefox, Opera, Microsoft Edge, Safari, Brave

Important Files and Folders

The basic structure of your website folder should have at least the following:

  • index.html (your homepage content)
  • images folder
  • styles folder
  • scripts folder
